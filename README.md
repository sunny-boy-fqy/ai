## 🚀 极速安装：一键开启智能时代

本工具采用“零污染”原则，自动为您配置私有 Node.js 与 Python 虚拟环境，绝不影响系统全局设置。

### Linux / macOS
```bash
curl -fsSL https://raw.githubusercontent.com/sunny-boy-fqy/ai/main/install.sh | bash
```

### Windows (PowerShell)
```powershell
irm https://raw.githubusercontent.com/sunny-boy-fqy/ai/main/install.ps1 | iex
```

## ✨ 核心卓越特性：为何选择 AI CLI？

### 1. 🧩 恐怖的自主进化能力 (MCP 生态)
AI 不再局限于我给它的几个工具。通过内置的 **`search_market`** 逻辑，AI 可以实时联网检索 GitHub 上的 **Awesome MCP** 列表（涵盖数千个插件）。
*   **自主安装**：如果您需要访问浏览器、操作 GitHub Issue 或查询地图，只需在对话中提出，AI 会**自行寻找安装命令**并调用 `install_plugin` 完成部署。
*   **即插即用**：安装完成后，环境会自动热重载，AI 在当前对话中即可获得新能力。

### 2. 📜 完备的“时间胶囊” (ai history)
所有的对话智慧都会被妥善保存在 `~/.ai/session/` 中。
*   **智能概括**：系统会自动提取第一轮对话的核心意图作为标题。
*   **无缝恢复**：随时运行 `ai history` 即可调取历史存档，输入编号继续之前的逻辑线。
*   **精简管理**：支持 `d编号` 快速销毁不需要的记忆。

### 3. 🛡️ 极致的环境隔离与安全性
*   **零污染**：内置绿色版 Node.js 和 Python venv，卸载时通过【自克隆技术】彻底抹除所有足迹。
*   **SSH 同步**：`ai download/update` 仅支持 SSH 协议，配合 `GIT_TERMINAL_PROMPT=0` 强制禁用密码弹出，利用您的私钥确保配置同步绝对安全。
*   **叠加模式**：多台设备同步时采用智能合并算法，API Key 自动查重并顺延编号，绝不覆盖已有数据。

## 📖 命令行全指令详解

### 💬 智力交互
*   **`ai [问题内容]`**: 闪电提问。支持长句输入，AI 将利用当前最强模型即时响应。
*   **`ai chat`**: 深度对话模式。
    *   输入 `"""`：开启/结束多行长文本粘贴模式。
    *   `-f [文件路径]`：挂载本地文件内容供 AI 分析。
*   **`ai history`**: 启动历史管理器。

### ⚙️ 矩阵配置
*   **`ai status`**: 查看当前运行矩阵（活跃商、模型、工作区、本地覆盖状态）。
*   **`ai list`**: 以可视化列表展示所有已录入的供应商、驱动类型及 Key 储备。
*   **`ai new`**: 边界扩展。引导式添加新 Key，支持自动识别已有供应商并保留 URL。
*   **`ai model`**: 管理模型池。切换版本、查看历史或录入从未出现过的新模型。
*   **`ai switch`**: 身份闪切。在 DeepSeek, OpenAI, Groq 等巨头服务间快速切换。
*   **`ai delete`**: 精准剔除。删除后编号自动重排，保持目录整洁。
*   **`ai workspace [路径]`**: 设置 AI 的活动领地，锁定文件操作工具的权限范围。

### ☁️ 云端同步与维护
*   **`ai download [SSH-URL]`**: 将云端配置下载并叠加合并到本地。
*   **`ai update [SSH-URL]`**: 将本地打磨好的 Key 矩阵一键叠加推送至远端。
*   **`ai upgrade [v]`**: 核心逻辑迭代。可指定版本号（如 v1.0）进行精准降级或升级。
*   **`ai uninstall`**: 毁灭性清理。杀掉进程、删除环境、清理 PATH，不留任何隐患。
